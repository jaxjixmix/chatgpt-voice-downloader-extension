How It Works

Three steps. That's it.

No configuration, no setup, no sign-up. Install and start downloading.

01

Install the extension

Download the zip, unpack it, and load it as an unpacked extension in Chrome. It only runs on chatgpt.com.

02

Click "Read Aloud"

Use ChatGPT's built-in Read Aloud on any response. The extension detects it automatically in the background.

03

Audio auto-downloads

The audio file saves to your machine automatically. No extra clicks — just check your downloads folder.

Features

Built for simplicity

No bloat. No tracking. Just the feature you need.

Automatic capture

Just click Read Aloud like you normally would. The extension captures the audio and downloads it automatically. Zero extra steps.

Privacy first

Everything happens locally in your browser. No data leaves your machine. No analytics, no telemetry, no servers.

AAC audio format

Audio is saved as high-quality AAC files that work everywhere — play offline, use in content, or share with anyone.

Open source

Fully open source on GitHub. Inspect the code, contribute, or fork it. Transparency by default.
